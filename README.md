# Systems of Linear Equations: Interactive Math Scavenger Hunt

An interactive web application for **solving systems of linear equations** using the **substitution method**. 
Perfect for Algebra 1 students, teachers, and anyone learning basic algebra.

## 🎯 Features
- **Step-by-step guidance** through the substitution method
- **Interactive graphing** to visualize solution as intersection point
- **Immediate feedback** on your answers
- **Progress tracking** (Score: 0/5)

## 🧮 Current Problem
Solve the system:
- 2x + 4y = 4
- 2x + y = 2

Solution: x = 0.67, y = 0.67
